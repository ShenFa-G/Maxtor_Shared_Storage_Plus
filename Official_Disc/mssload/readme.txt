1. Copy MSSLoad directory to your local HDD directory.
2. Go to command prompt.
3. Change directory to MSSLoad
4. Execute the application "nasload.exe"
5. The application will wait for any Maxtor Shared Storage unit which need to reload the kernel.
6. Once the application finds the Shared Storage unit, It will prompt for admin user name and password.
7. Type-in correct admin user name and password. The application will upload kernel to Shared Storage unit.
8. Wait until the Shared Storage unit LED becomes green.
9. Reset the unit by inserting a paper clip into the reset hole, and hold for 5 seconds.
10. Current kernal version is 2.3.1

1. Copiez le répertoire MSSLoad dans votre répertoire HDD local.
2. Accédez à l’invite de commande.
3. Faites passer le répertoire à "MSSLoad".
4. Exécutez l’application "nasload.exe".
5. L’application attend alors toute unité Maxtor Shared Storage devant recharger le noyau.
6. Le mot de passe et le nom d’utilisateur de l’administrateur vous sont demandés dès que l’application a trouvé l’unité Shared Storage.
7. Saisissez le nom d’utilisateur et le mot de passe. L’application télécharge alors le noyau vers l’unité Shared Storage.
8. Patientez jusqu’à ce que le voyant lumineux de l’unité Shared Storage devienne vert.
9. Réinitialisez l’unité en insérant un trombone dans le trou "reset" et en le maintenant enfoncé pendant 5 secondes.
10. La version actuelle du noyau est 2.3.1

1. Copiare la directory di caricamento Maxtor Shared Storage sulla directory locale dell'unità disco rigido.
2. Spostarsi sul prompt dei comandi.
3. Sostituire la directory locale con la directory di caricamento Maxtor Shared Storage
4. Eseguire l'applicazione denominata "nasload.exe"
5. L'applicazione attenderà che vengano individuate tutte le unità Maxtor Shared Storage che necessitano del caricamento del kernel.
6. Una volta che l'applicazione ha trovato l'unità Shared Storage, verranno richiesti nome utente e password dell'amministratore.
7. Inserire il nome utente e la password dell'amministratore corretti. L'applicazione caricherà a questo punto il kernel sull’unità Shared Storage.
8. Attendere che il LED dell'unità Shared Storage diventi verde.
9. Reimpostare l'unità inserendo una graffetta nel foro di ripristino e tenendo premuto per 5 secondi.
10. L'attuale versione kernel è la 2.3.1

1. Kopieren Sie das Verzeichnis MSSLoad in Ihr lokales Festplattenverzeichnis.
2. Gehen Sie zur Eingabeaufforderung.
3. Wechseln Sie in das Verzeichnis MSSLoad.
4. Führen Sie die Anwendung "nasload.exe" aus.
5. Die Anwendung wartet, wenn ein Maxtor Shared Storage-Gerät den Kernel neu laden muss.
6. Sobald die Anwendung das Shared Storage-Gerät gefunden hat, werden Sie dazu aufgefordert, den Benutzernamen und das Kennwort des Administrators einzugeben.
7. Geben Sie den Benutzernamen und das Kennwort ein. Daraufhin lädt die Anwendung den Kernel auf das Shared Storage-Gerät hoch.
8. Warten Sie, bis die Leuchtanzeige des Shared Storage-Geräts grün wird.
9. Setzen Sie das Gerät zurück, indem Sie eine Büroklammer für 5 Sekunden in die Rücksetzöffnung stecken.
10. Die aktuelle Kernelversion ist 2.3.1

1. Copie el directorio MSSLoad al directorio de su unidad de disco duro.
2. Vaya al indicador de comandos.
3. Cambie el directorio a MSSLoad.
4. Ejecute la operación "nasload.exe".
5. La aplicación esperará a que todas aquellas unidades Maxtor Shared Storage que precisen recargar el núcleo lo hagan.
6. Una vez haya encontrado la unidad Shared Storage, la aplicación pedirá el nombre y contraseña del administrador.
7. Escriba la contraseña y el nombre de usuario del administrador. La aplicación cargará el núcleo a la unidad Shared Storage.
8. Espere hasta que el indicador luminoso de la unidad Shared Storage cambie a verde.
9. Reinicie la unidad insertando un clip en el orificio de reinicio y manteniéndolo apretado durante 5 segundos.
10. La versión del núcleo es la 2.3.1
